# I18N
## 本地化关注地方
- 姓名规则，比如是姓氏在前还是名字在前
- 电话号码格式
- 日期和时间格式，如日/月/年、月/日/年
- 货币，可能需要转换成对应的当地货币值
- 书写方向，大多数语言是从左向右，但希伯来语和阿拉伯语是从右向左，有些亚洲语言是竖着书写
- 度量单位，如公制或英制
- 标点，例如各语言使用的引号各不相同，在英语中是 (`""`)，在德语中是 (`„“`)，在法语中是 (`<<>>`)
- 符号和象形图，包括勾号、停止标志以及用颜色传达的意思
- 电压、频率和插头
- 法律要求，如使用欧盟公民的个人数据时应遵守的 GDPR(General_Data_Protection_Regulation)
