## 02 ｜理解 Hooks：React 为什么要发明 Hooks

- class 不合适
  - 组件无继承，多为组合
  - 组件比较常不去访问组件实例
- function：功能满足
  - 组件没法保存状态
- hooks
  - 保存状态
  - 有生命周期
- hooks 实现浅析：在函数组件的外部开辟一个空间来保存组件的状态

## 03 ｜内置 Hooks（1）：如何保存组件状态和使用生命周期

- useState: 保存组件的状态
  - 1. 只影响 UI；
  - 2. 可以通过计算出的都不能当做状态；
  - 3. 状态改变会引起 UI 重新渲染
- useEffect: 函数式组件执行完毕后，再执行
